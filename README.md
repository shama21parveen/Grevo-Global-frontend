# Grevo Global Frontend (intership task)

This project is a frontend website built for Grevo Global. It is developed using React and Vite, with a focus on creating a clean, modern, and responsive user interface.

## Features

- Responsive frontend design
- Reusable React components
- Clean project structure
- Fast development setup with Vite

## Tech Stack

- React
- Vite
- JavaScript
- CSS
